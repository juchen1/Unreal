- Mouse Left -> 전후[평행이동] 좌우[회전]
- Mouse Right -> 현 위치에서 360도 회전

- Keyboard WASD[with mouse] -> 상하좌우 이동(마치 캐릭터 처럼)
- Keyboard Q E[with mouse] -> 위 아래 이동

### Contents 이동하기(의자)
- W[이동] E[회전] R[스케일]
- X[red] Y[green] Z[blue]
- 한 축, 두축 혹은 전체 축을 선택해서 작동가능
